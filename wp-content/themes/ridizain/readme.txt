== Ridizain ==

Ridizain - Pronounced "Redesign" is a redesigned version of the default theme Twenty Fourteen.
Build Magazine type sites with bold picture statements both in the blog feed & widget areas.

Features: 
Custom Recent Post Widgets with three widget areas to boot - Content Sidebar, Secondary Sidebar, 
and Footer widget area. Featured posts in a grid or slider format on the front page,
slider options fade or slide, upload logo image and much more.

Coded by Zulfikar Nore
Brought to you by WP Strap Code, http://wpstrapcode.com/

Requires at least:	3.6.0
Tested up to:		3.9-alpha
Stable tag:			1.0.19
